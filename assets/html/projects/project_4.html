• Brawlhalla

    Game Development Design

    Project Overview

    The Brawlhalla project is a school assignment focused on game development and design. Our goal is to create an engaging and visually appealing game that provides a fun experience for players.

    Key Features

    • Multiplayer support
    • Customizable characters
    • Various game modes
    • High-quality graphics and sound

    Technical Details

    We developed the game using Unity, leveraging its powerful engine for both 2D and 3D game development. The game assets were created using Blender and Photoshop.

    Implementation Challenges

    One of the primary challenges was ensuring smooth gameplay and minimizing latency in multiplayer mode. We optimized the network code and conducted extensive testing to achieve this.

    Future Developments

    Future developments include adding new game modes, expanding the character roster, and integrating community feedback to improve the overall experience.
